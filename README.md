# Exercism-tracks – Solutions avec contexte

Ce dépôt regroupe une sélection d’exercices issus de la plateforme [Exercism](https://exercism.org).

J’y conserve tout le contexte nécessaire pour rejouer les exercices (tests, structure du projet, consignes), ainsi que mes solutions personnelles.

L'objectif est de pouvoir facilement revenir sur une solution et tester différentes approches d'implémentation.

##  Organisation du dépôt

Chaque exercice est regroupé par langage, puis par nom d’exercice.

### Ce qui vient d’Exercism

- l’énoncé de chaque exercice ;
- la structure des projets ;
- les tests automatiques ;
- parfois certains fichiers d’initialisation.

### Ce qui vient de moi

- les implémentations dans les dossiers `src/`

## Pourquoi conserver ces exercices ?

Même des exercices “simples” peuvent être utiles pour :

- tester un langage ou un écosystème ;
- s’exercer à écrire du code lisible, documenté et testé ;
- expérimenter différentes stratégies de résolution ;
- revenir plus tard et tenter une version plus élégante.
